== NAME

nng_msg_free - free a message

== SYNOPSIS

[source, c]
----
#include <nng/nng.h>

void nng_msg_free(nng_msg *msg);
----

== DESCRIPTION

The `nng_msg_free()` function deallocates the message _msg_ entirely.

== RETURN VALUES

None.

== ERRORS

None.
