== NAME

nng_msg_clear - clear message body content

== SYNOPSIS

[source, c]
----
#include <nng/nng.h>

void nng_msg_clear(nng_msg *msg);
----

== DESCRIPTION

The `nng_msg_clear()` function resets the body length of _msg_ to zero.

== RETURN VALUES

None.

== ERRORS

None.
