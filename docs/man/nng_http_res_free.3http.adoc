== NAME

nng_http_res_free - free HTTP response structure

== SYNOPSIS

[source, c]
----
#include <nng/nng.h>
#include <nng/supplemental/http/http.h>

void nng_http_res_free(nng_http_res *req);
----

== DESCRIPTION

The `nng_http_res_free()` function deallocates the HTTP response structure
_res_ entirely.

== RETURN VALUES

None.

== ERRORS

None.
