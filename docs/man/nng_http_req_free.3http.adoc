== NAME

nng_http_req_free - free HTTP request structure

== SYNOPSIS

[source, c]
----
#include <nng/nng.h>
#include <nng/supplemental/http/http.h>

void nng_http_req_free(nng_http_req *req);
----

== DESCRIPTION

The `nng_http_req_free()` function deallocates the HTTP request structure
_req_ entirely.

== RETURN VALUES

None.

== ERRORS

None.
