nng_url_free(3)

NAME

nng_url_free - free a URL structure

SYNOPSIS

#include <nng/nng.h>

void nng_url_free(nng_url *url);

DESCRIPTION

The nng_url_free() function deallocates the url entirely, including any of its members.

RETURN VALUES

None.

ERRORS

None.
