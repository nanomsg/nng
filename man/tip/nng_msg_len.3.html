nng_msg_len(3)

NAME

nng_msg_len - return message body length

SYNOPSIS

#include <nng/nng.h>

size_t nng_msg_len(nng_msg *msg);

DESCRIPTION

The nng_msg_len() returns the length of the body of message msg.

RETURN VALUES

Length of message body.

ERRORS

None.
