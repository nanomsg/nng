NAME

nng_http_res_get_status - return HTTP status code

SYNOPSIS

#include <nng/nng.h>
#include <nng/supplemental/http/http.h>

uint16_t nng_http_res_get_status(nng_http_res *res);

DESCRIPTION

The nng_http_res_get_status() returns a numeric code corresponding to the HTTP status of the response res.

For convenience, a number of predefined symbols corresponding to well-known HTTP status codes are available.

When displaying status information to users (or logging such information), consider also including the reason obtained with nng_http_res_get_reason().

RETURN VALUES

HTTP status code.

ERRORS

None.
