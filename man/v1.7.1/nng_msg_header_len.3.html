nng_msg_header_len(3)

NAME

nng_msg_header_len - return message header length

SYNOPSIS

#include <nng/nng.h>

size_t nng_msg_header_len(nng_msg *msg);

DESCRIPTION

The nng_msg_header_len() returns the length of message header of msg.

RETURN VALUES

Length of message header.

ERRORS

None.
